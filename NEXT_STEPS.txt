KEYINGI QADAMLAR:
=================================================================

1. SHAXSIY MA'LUMOTLARNI YANGILASH (30 daqiqa)
   - README.md faylida [Your Name] ni o'zgartiring
   - src/*.html fayllarida shaxsiy ma'lumotlarni kiriting
   - Kontakt ma'lumotlarini yangilang

2. GITHUB'GA YUKLASH (10 daqiqa)
   a) GitHub'da yangi repository yarating:
      Nomi: devops-portfolio-[sizning-ismingiz]
      Public qiling
   
   b) Quyidagi buyruqlarni bajaring:
      git remote add origin https://github.com/[username]/devops-portfolio-[yourname].git
      git push -u origin main
      git push origin --all

3. GITHUB PAGES NI YOQISH (5 daqiqa)
   - Repository Settings > Pages
   - Source: gh-pages branch
   - Save

4. MONITORING O'RNATISH (20 daqiqa)
   - Google Analytics yarating
   - UptimeRobot sozlang
   - docs/MONITORING_SETUP.md ni o'qing

5. DOCKER HUB (15 daqiqa)
   docker build -t devops-portfolio .
   docker login
   docker tag devops-portfolio [username]/devops-portfolio:latest
   docker push [username]/devops-portfolio:latest

6. DALILLAR TO'PLASH (1 hafta)
   - 15+ screenshot oling
   - 3 ta video yozing
   - Monitoring ma'lumotlarini to'plang
   - Performance testlarini o'tkazing

7. HISOBOTLAR YOZISH (2-3 kun)
   - docs/REPORT_TEMPLATE_*.md fayllaridan foydalaning
   - Jami: 2300-2500 so'z

8. TOPSHIRISH (1 soat)
   - Evidence/ papkasiga screenshotlar
   - Reports/ papkasiga PDF fayllar
   - links.txt ni yangilang
   - ZIP fayl yarating
